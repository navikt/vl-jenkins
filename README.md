vl-jenkins
================

Jenkins pipeline scripts for team familie.

# Komme i gang

På jenkins, konfigurer til å bygge med Jenkinsfile. I prosjekt, importer library i Jenkinsfile og kall ønsket pipeline-script. 

Eksempel: 

`@Library('vl-jenkins')`

`naisPipeline()`
